# SmartBrain FrontEnd

Smartbrain is a final project for ZTM course

## Description

A web application that calls Clarifai API to detect faces in images. Input an image url, and the app will detect faces in the image.

### Executing program

1. Clone repo

2. Run
```
npm install
```

3. Run 
```
npm start
```
